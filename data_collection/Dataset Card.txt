## **3. Data Collection Method**

The dataset was collected programmatically using the following reproducible Python code:

```python
import yfinance as yf

nasdaq = yf.download("^IXIC",
                     start="2020-01-01",
                     end="2025-01-01")

nasdaq["return"] = nasdaq["Close"].pct_change()
nasdaq = nasdaq.dropna()

nasdaq.to_csv("nasdaq_dataset.csv")
```

This ensures transparency and full reproducibility for team members and instructors.

## **4. Dataset Structure and Features**

The downloaded dataset contains the following columns:

| Column        | Description                                      |
| ------------- | ------------------------------------------------ |
| **Date**      | Trading date (index)                             |
| **Open**      | Opening price                                    |
| **High**      | Highest price of the day                         |
| **Low**       | Lowest price of the day                          |
| **Close**     | Closing price                                    |
| **Adj Close** | Adjusted closing price (split/dividend adjusted) |
| **Volume**    | Total trading volume                             |
| **return**    | Daily return computed as `Close.pct_change()`    |

These features support downstream tasks such as return prediction, volatility estimation, and backtesting.
